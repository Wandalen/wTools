# Documentation

## Scope

Design and API documentation for `willbe2`.

`willbe2` is a pure re-export alias for `willbe`. It exposes the identical library API (`pub use ::willbe::*`) and delegates all binary CLI commands to `willbe::run()`. All design documentation, architectural patterns, and feature specifications live in the upstream crate: [`willbe/docs/`](../../willbe/docs/readme.md).
