# Feature Doc Entity

### Scope

- **Purpose**: Provides navigational hubs that cross-reference all artifacts for each user-facing capability.
- **Responsibility**: Indexes all feature doc instances in the `feature/` collection.
- **In Scope**: Feature scope, design decisions, and cross-references to source, test, doc, and task artifacts.
- **Out of Scope**: Content duplicated from pattern/, invariant/, or api/ — feature/ links only, never re-states.
