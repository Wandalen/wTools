# Docs

## Scope

Design and API documentation for `derive_tools_meta`. Organized as typed doc entity instances: features describe what the crate does, invariants define behavioral contracts, API docs reference the public interface, and integration docs describe cross-crate relationships.

See [entities.md](entities.md) for the complete registry of all doc instances.
