# Examples

Runnable usage examples for `strs_tools_meta` procedural macros.

## Responsibility Table

| File | Responsibility |
|------|----------------|
| `split_basic.rs` | Demonstrate basic `optimize_split!` macro usage patterns |
| `match_basic.rs` | Demonstrate basic `optimize_match!` macro usage patterns |

## Running Examples

```bash
# Run the split example
cargo run --example split_basic --features optimize_split

# Run the match example
cargo run --example match_basic --features optimize_match
```
