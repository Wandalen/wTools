# tests/docs

Test surface spec registry for `strs_tools_meta` procedural macros.

## Scope

- **In Scope**: Test spec files covering `optimize_split!` and `optimize_match!` macro behavior, API contracts, and invariants.
- **Out of Scope**: Implementation test files (see parent `tests/` directory); manual testing plans (see `manual/`).
- **Status**: Active.

## Responsibility Table

| File / Dir | Responsibility |
|------------|----------------|
| `feature/` | Feature behavior test specs |
| `api/` | API contract test specs |
| `invariant/` | Invariant enforcement test specs |
