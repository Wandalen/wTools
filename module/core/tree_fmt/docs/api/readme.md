# API Reference

## Purpose
Public API surface documentation for tree_fmt library types, builders, configurations, and formatters.

## Responsibility Table

| File | Responsibility |
|------|----------------|
| `data_types.md` | Core data structures: TreeNode, TableView, ColumnData, TreeSymbols |
| `builders.md` | Construction APIs: RowBuilder, TreeBuilder, FlattenConfig |
| `config_types.md` | Configuration types: TableConfig, ExpandedConfig, TreeConfig and enums |
| `formatters.md` | Formatter APIs: all formatters, Format trait, TableShapedFormatter trait |
