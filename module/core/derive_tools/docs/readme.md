# Docs

## Scope

Design and API documentation for `derive_tools`. Organized as typed doc entity instances: features describe what the crate does, invariants define behavioral contracts, and API docs reference the public interface.

See [entities.md](entities.md) for the complete registry of all doc instances.

| [integration/](integration/readme.md) | Integration documentation — dependencies and consumers |
