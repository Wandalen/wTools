# CLI Test Specs

### Scope

Test spec files for `docs/cli/` command, parameter, and parameter group documentation.

### Responsibility Table

| Directory | Responsibility |
|-----------|----------------|
| command/ | Spec files for each CLI command group (IT- prefix) |
| param/ | Spec files for CLI parameters (EC- prefix) |
| param_group/ | Spec files for CLI parameter groups (CC- prefix) |
