# config/

Formatter parameter types, split by single responsibility.

| File | Responsibility |
|------|----------------|
| mod.rs | Public re-exports from all sub-modules |
| tree_config.rs | `TreeConfig` and `TreeSymbols` types |
| table_enums.rs | Configuration enums for `TableConfig`: variants, separators, and styles |
| table_caption.rs | `TableCaption` and caption constants |
| table_config.rs | `TableConfig` and `PaddingSide` types |
| expanded_config.rs | `ExpandedConfig` type |
