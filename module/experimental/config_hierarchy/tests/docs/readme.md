# Test Surface Docs

### Scope

- **Purpose**: Provide test specifications for all `docs/` entity instances.
- **Responsibility**: One spec file per `docs/` instance; cases cover all documented behaviors.
- **In Scope**: Test specs for algorithm, api, feature, format, invariant, and pattern doc entities.
- **Out of Scope**: Implementation tests (→ `tests/*.rs`); manual testing plans (→ `tests/manual/`).

### Responsibility Table

| Name | Purpose |
|------|---------|
| `algorithm/` | Test specs for `docs/algorithm/` instances |
| `api/` | Test specs for `docs/api/` instances |
| `feature/` | Test specs for `docs/feature/` instances |
| `format/` | Test specs for `docs/format/` instances |
| `invariant/` | Test specs for `docs/invariant/` instances |
| `pattern/` | Test specs for `docs/pattern/` instances |
