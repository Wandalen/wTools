# module/experimental/

Workspace crates in active development. Included in workspace via `module/experimental/*` glob.

| Directory | Responsibility |
|-----------|----------------|
| `asbytes/` | POD data to byte slices via bytemuck-backed traits. |
| `async_from/` | Async conversion traits and derive macros. |
| `benchkit/` | Benchmarking framework with criterion integration. |
| `component_model/` | Component model derive macros. |
| `component_model_meta/` | Procedural macros for `component_model`. |
| `component_model_types/` | Types and traits for `component_model`. |
| `config_hierarchy/` | Hierarchical configuration resolution with file discovery. |
| `deterministic_rand/` | Hierarchical seeded RNG with switchable determinism. |
| `for_each/` | Enhanced for-each iteration macros. |
| `implements/` | Compile-time trait implementation checks. |
| `include_md/` | Include Markdown files in doc comments. |
| `interval_adapter/` | Range normalization traits and non-iterable intervals. |
| `mem_tools/` | Memory layout inspection utilities. |
| `multiline_input/` | Terminal multiline input with interactive editing. |
| `program_tools/` | Run Rust files and projects as scripts. |
| `reflect_tools/` | Runtime reflection for struct fields. |
| `reflect_tools_meta/` | Procedural macros for `reflect_tools`. |
| `test_tools/` | Test harness with standalone build support. |
| `unitore/` | Feed aggregation and storage engine. |
| `wca/` | CLI command aggregation framework. |
| `willbe/` | Workspace publish and consistency tool. |
