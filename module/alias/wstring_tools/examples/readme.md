# Examples Directory

Example programs demonstrating wstring_tools functionality.

## Organization

| File | Responsibility |
|------|----------------|
| wstring_toolst_trivial_sample.rs | Basic string splitting demonstration |

## Usage

Run examples with:
```bash
cargo run --example wstring_toolst_trivial_sample --all-features
```

## Notes

All examples demonstrate wstring_tools API (not strs_tools directly) to showcase the alias crate's usage patterns.
